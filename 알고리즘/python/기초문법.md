# 2. 변수입력과 연산자

<pre><code>a, b=input("숫자를 입력하세요 : ").split() 
print(a, b)
print(a+b) # 입력자에서 출력되는 값은 string형임
a=int(a)
b=int(b)
print(a+b) # 5출력됨
</pre></code>

따로 int()를 하지 않아도 map으로 원하는 변수 형태를 지정할 수 있음
그리고 
<pre><code>a, b=map(int, input("숫자를 입력하세요 : ").split())
print(a+b)
